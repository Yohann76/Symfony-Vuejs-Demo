"# Symfony-Vuejs-Demo" 


Start project : 

    $ git clone 
    $ composer install
    $ yarn install 

Compilation : 

    $ yarn encore dev // for dev
    $ yarn encore production // for prod 
    $ symfony server:start // launch symfony serve 


This application use : 
    - Symfony
    - VueJS
    - Webpack Encore
    - Tailwind CSS
    - Ansible
    - Docker
    - Github page
